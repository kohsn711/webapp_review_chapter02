# 第2章: HTML / CSS

## 目的

- HTML の基本構造を理解する
- フォーム画面と一覧画面を作る
- CSS で見た目を整える

## 題材

- ゲームの点数記録ページ

## 画面構成

- `index.html`
  - 点数を登録するページ
- `records.html`
  - 登録済みスコアを表で見るページ
- ナビゲーションバー
  - 2つのページを移動するためのリンク

## フォーム項目

- プレイヤー名
- 点数
- プレイ日
- メモ

## 使うファイル

- `starter/index.html`
- `starter/records.html`
- `starter/style.css`
- `starter/score-header.svg`
- `reference/page-spec.md`
- `reference/html-guide.md`
- `reference/css-guide.md`

## 補足

- この章では JavaScript を使わず、保存処理は作らない
- 一覧は `records.html` に固定データを直接書いている
- `a href` はページ移動の例として使う
- ヘッダー画像は HTML に直接書かず、CSS の `background-image` で表示している
