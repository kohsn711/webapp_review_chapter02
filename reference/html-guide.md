# HTML解説メモ

## HTMLの役割

HTML は、ページに何を置くかを決めるための言語です。

- 見出し
- 文章
- 入力欄
- ボタン
- 表

といった画面の骨組みを作ります。

## ファイルの最初にあるもの

### `<!doctype html>`

このファイルが HTML 文書であることをブラウザに伝えます。

### `<html lang="ja">`

ページ全体を囲むタグです。`lang="ja"` は、このページが日本語で書かれていることを表します。

### `<head>`

画面には直接見えない設定を書く場所です。

この中では主に次のものを書いています。

- `<meta charset="UTF-8" />`
  - 文字コードの指定です
- `<meta name="viewport" ... />`
  - スマホでも見やすくするための設定です
- `<title>ゲームの点数記録</title>`
  - ブラウザのタブに表示されるタイトルです
- `<link rel="stylesheet" href="./style.css" />`
  - CSS ファイルを読み込む指定です

### `<body>`

実際に画面に表示される内容を書く場所です。

## この章のページ構成

### `<main class="page">`

ページの主要な中身をまとめるタグです。`class="page"` は CSS を当てるための目印です。

## 見出しエリア

### `<header class="page-header">`

ページのタイトル付近をまとめています。

### `<h1>`

ページ全体でいちばん大きな見出しです。

### `<p>`

説明文を書くためのタグです。

今回は画像ファイル自体は HTML に直接書かず、CSS の `background-image` でヘッダーの背後に表示しています。

## ナビゲーション

### `<nav class="site-nav">`

ページ移動のリンクをまとめるタグです。

今回は、見出しの下にナビゲーションバーを置いています。

### `<a href="...">`

リンクを作るためのタグです。

今回のファイルでは、登録ページと一覧ページを移動するために使っています。

```html
<a href="./records.html">スコア一覧</a>
```

- `href`
  - どこへ移動するかを指定します
- `./records.html`
  - 同じフォルダにある `records.html` へ移動します

## セクションの分け方

### `<section class="panel">`

画面を意味のあるかたまりごとに分けるためのタグです。

登録ページには、次のまとまりがあります。

- 見出しエリア
- ナビゲーションバー
- 点数を登録するエリア

一覧ページでは、見出しエリアとナビゲーションバーの下にスコア一覧の表を置いています。

## フォームまわり

### `<form>`

入力欄やボタンをまとめるためのタグです。

この章ではまだ保存処理を作らないので、見た目の土台として使っています。

### `<label>`

入力欄の名前と入力欄をセットで扱うためのタグです。

たとえば「プレイヤー名」という文字と、その入力欄を一緒にまとめています。

### `<input>`

1 行だけ入力するための部品です。

今回使っているもの:

- `type="text"`
  - 文字を入力する
- `type="number"`
  - 数字を入力する
- `type="date"`
  - 日付を入力する

### `<textarea>`

複数行のメモを書くための入力欄です。

### `<button>`

ボタンを表示するタグです。

今回はまだ JavaScript や保存処理を付けていないので、見た目だけのボタンです。

## 表の部分

### `<table>`

表全体を作るタグです。今回は `records.html` で使っています。

### `<thead>`

表の見出し行をまとめる場所です。

### `<tbody>`

表のデータ本体をまとめる場所です。

### `<tr>`

表の 1 行を表します。

### `<th>`

表の見出しのセルです。

### `<td>`

表のデータのセルです。

## 今回の HTML の読み方

`starter/index.html` は、上から順に次の流れで読むと分かりやすいです。

1. ファイル全体の設定を見る
2. 背景画像付きの見出しとページタイトルを見る
3. ナビゲーションバーを見る
4. 入力フォームを見る

`starter/records.html` は、見出しとナビゲーションバーのあとに固定データの表を置いたページです。`a href` を使って、登録ページと行き来できます。

## この章でまだやらないこと

- データを保存する
- ボタンを押したときに動かす
- 一覧を自動で増やす

この章では、まず HTML でページの骨組みとリンクのつながりを作ることに集中します。
