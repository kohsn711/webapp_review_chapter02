# 画面仕様メモ

## 画面の役割

- ゲームのプレイ結果をあとで見返せるように記録する

## 登録ページの必須要素

- ページタイトル
- ヘッダー背景画像
- ナビゲーションバー
- 点数登録フォーム

## 一覧ページの必須要素

- ページタイトル
- ヘッダー背景画像
- ナビゲーションバー
- スコア一覧テーブル

## フォーム項目

- プレイヤー名
- 点数
- プレイ日
- メモ

## 一覧の列

- プレイヤー名
- 点数
- プレイ日
- メモ

## 注意

- この章では保存処理は作らない
- 一覧データは HTML に直接書いた固定データを使う
- ナビゲーションバーは常に表示し、リンクでページ移動できる
- スマホではナビゲーションバーを画面下部に固定する
- 見た目と構造を作ることに集中する
